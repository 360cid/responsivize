Responsivize — responsive table plugin

Responsivize is a table that fits on your screen.

How to Use

Initialization:

You call it like this...

$('#myTable').responsivize();

Options:

highlightSortedColumn { boolean }
default: false
set a background color on the currently sorted column
modal { boolean }
default: true
display the column selector in a modal or dropdown. Can be convenient for progressive enhancement, e.g. touch events vs no touch events.
columnPriority
default: TODO (it's complicated)
Order in which columns should hide. Most of this is handled via css; script is required to handle window resizing and various sort options
